# Vite Vanilla Todo

This project is a demo to understand how vite is used to make projects, it uses vanilla JS to create a todolist app

#### Getting Started

```bash

# clone this project
git clone <this project url>

# install dev packages
npm install

# start dev server
npm run dev

```
